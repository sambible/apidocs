GET /katello/api/ping
Shows status of Katello system and it's subcomponents

This service is only available for authenticated users

Examples

GET /katello/api/ping
200
{
  "status": "ok",
  "services": {
    "candlepin": {
      "status": "ok",
      "duration_ms": "23"
    },
    "candlepin_auth": {
      "status": "ok",
      "duration_ms": "0"
    },
    "foreman_tasks": {},
    "katello_events": {},
    "candlepin_events": {},
    "pulp3": {},
    "pulp3_content": {},
    "pulp": {
      "status": "ok",
      "duration_ms": "38"
    },
    "pulp_auth": {
      "status": "ok",
      "duration_ms": "0"
    }
  }
}

Params

Param name Description
